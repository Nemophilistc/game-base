## 本地运行

如果需要在本地运行（不依赖网络）：

```bash
# 1. 克隆仓库
git clone https://github.com/Nemophilistc/game-base.git

# 2. 进入目录
cd game-base

# 3. 启动HTTP服务器
python -m http.server 8000

# 4. 打开浏览器访问
# http://localhost:8000
```

或者直接双击 `启动游戏服务器.bat`（Windows）。

## 技术栈

- **HTML5 Canvas** — 游戏渲染
- **原生 JavaScript (ES Modules)** — 无框架依赖
- **Web Audio API** — 音效合成（无外部音频文件）
- **localStorage** — 最高分/存档持久化

## 设计原则

- 每款游戏独立目录：`GameName/index.html` + `GameName/js/` 模块
- 暗色毛玻璃UI主题，中文界面
- 所有游戏支持键盘操作，部分支持鼠标/触屏
- 零依赖，无需 npm/node
